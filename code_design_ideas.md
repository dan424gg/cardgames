## Code Design Ideas

#### Reusable Components

* **Title**
  * Luckiest Guy font
  * Text

* **Card (abstract)**
  * Icon
  * Text
  * Background color

* **Action Card (card)**
  * Icon
  * Text
    * Background color -> green / white
  * Dropdown (optional)
    * Children views
    * On tap, show dropdown
  * Action (optional)
    * Action
    * On tap, take action
  * Toggle action (optional)
    * Action

* **Section**
  * Header (card)
    * Icon
    * Title
    * Background color
  * Children

* **Button**
  * Color
  * Text
  * Action

#### Colors

* **Primary:** D7EDD7
* **Secondary:** FFFFFF
* **Tertiary:** C6C6C8
* **Background:** F0F7F0
* **Bad:** F4DBD4

#### Main Button

* Icon
* Title
* Color override
* Dropdown sub-view

#### Sub Button

* Optional Icon
* Title
* Dropdown sub-view
  * Either redirect to subview (show arrow) or toggle

#### Back Button

* Arrow icon
* Redirect to previous view

#### Section

* Icon
* Title
* Children views
